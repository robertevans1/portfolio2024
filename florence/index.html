Florence

Flutter mobile app

From April 2022 until November 2023 I worked as a developer at Florence. Initially I worked solely as a Flutter developer working on their mobile app. Later as a full stack developer or product engineer. The florence app is used by over 100k care professionals across the UK, France and Canada. I was one of 3 mobile developers at the company and so had a significant impact on the app.

Technical notes

The mobile app is built using Flutter, the backend is written using ruby on rails. There is a web app used by care homes with a frontend written in React. Ultimately I worked on all 3 codebases. The app uses riverpod for state management, and a number of modern packages for handling event tracking, analytics, authorisation and error handling. Some of the features I had responsibility for in the app included:

  • Adding a new flow for care professionals to claim compensation for miles driven
  • Several updates to the onboarding pipeline
  • Adding a process to use machine learing tools as a first pass assessing uploaded documentation
  • Adding the ability for care professionals to work 'compound' shifts, where they stay overnight at a residence
