Image Processing App

Flutter frontend, C++ backend

As I've worked both as a mobile and a c++ engineer I wanted to showcase how the two can be combined. This project is a simple image processing app that allows the user to blur an image. The app is built using flutter, and the image processing is done using a c++ backend. Creating a more complex image processing pipeline is something better suited to the c++ world. So combining the two allows for a more powerful app.
